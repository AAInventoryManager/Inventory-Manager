# Inventory Management

Feature ID: inventory.inventory.core

## Overview
Core inventory management covers viewing, adding, and editing items along with quantity-on-hand adjustments.

## Tier Availability
- Starter
- Professional
- Business
- Enterprise

## How It Works
- Items are listed in a single table with inline editing.
- Quantity-on-hand changes update inventory immediately.
- Changes sync across team members in real time.

## Limitations
- No approvals or multi-step workflows.
- Quantity changes are immediate and irreversible unless restored via trash.
- Item names must remain unique within a company.

## Support Expectations
Users may ask about quantity adjustments and item naming rules. Support should confirm that edits apply immediately.
