# Inventory Batch Edit

Feature ID: inventory.inventory.batch_edit

## Overview
Batch edit lets users update multiple items at once for fields like name, description, or quantity.

## Tier Availability
- Starter
- Professional
- Business
- Enterprise

## How It Works
- Select multiple items in the inventory table.
- Choose a field to update and confirm the batch change.
- All selected items are updated in a single action.

## Limitations
- Batch edits apply the same value to every selected item.
- Some fields may require confirmation before applying.

## Support Expectations
Users may request help undoing unintended bulk edits. Support should direct users to use the trash restore flow when applicable.
